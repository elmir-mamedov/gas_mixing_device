PROGRAM PRO MĚŘENÍ SKUTEČNÉHO OBJEMU STŘÍKAČKY

Program zcela naplní stříkačku a čeká, až uživatel zadá jakýkoli vstup prostřednictvím monitoru. 

Poté vytlačí objem plynu odpovídající deseti otáčkám pístového šroubu, zastaví se a čeká na další vstup. 

Vytlačování se opakuje každých deset otáček, dokud se je stříkačka zcela vyprázdněna.
